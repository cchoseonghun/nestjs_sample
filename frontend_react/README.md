# Sample Project - Client Server with React

## Usage

### `git glone 후 세팅`

```
npm i
```

### `React 실행`

```
npm run start
```

### `React 빌드`

```
npm run build
```

### `Express를 통해 빌드된 React로 서버 실행`

```
npm run dev
```

> 현재 간단한 게시글 불러오는 기능만 존재.   
> DB를 붙이지 않아 요청 시 [] 빈 array만 불러오는데   
> 추후 DB 연결되면 해결될 것으로 보임
